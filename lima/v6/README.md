# V6 OBD Reader Application

This folder contains the V6 OBD reader application for the Lima project.

## Structure
- `v6/` - Main application package
- `logging/` - Logging utilities
- `__init__.py` - Module initialization

## Features
- Modular design for OBD reading functionality
- Integrated logging support

## Usage
Import the `v6` module and use its API to interact with OBD devices.
